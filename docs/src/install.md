# Installing LWK

LWK is available for several languages.

## Rust
You can use the crates released on [crates.io](https://crates.io)

```rust,ignore
[dependencies]
lwk_wollet = "0.11.0"
lwk_signer = "0.11.0"
lwk_common = "0.11.0"
```

## Python
You can use the official python package: [lwk](https://pypi.org/project/lwk/)

```shell,ignore
pip install lwk
```

## Javascript/Typescript (Wasm)

### Browser / bundler

```shell,ignore
npm install @blockstream/lwk-web
```

```typescript,ignore
import * as lwk from "@blockstream/lwk-web";
```

### Node.js

```shell,ignore
npm install @blockstream/lwk-node
```

```typescript,ignore
import * as lwk from "@blockstream/lwk-node";
```
